|------------|----------------|---------------|

| `false`    | `false`        | False         |
| `0`        | `0`            | False         |
| Empty String | `""`        | False         |
| `null`     | `null`         | False         |

| Objects   | `{}`, `[]`     | True          |
| Functions | `function() {}`| True          |
